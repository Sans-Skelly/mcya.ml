    • How you can contribute to the development of MCYA.ML

    As a side-project, I have always wanted to run a large information place where you can formulate a great minecraft server and a place where yaml files are located.

    .

    Sadly, I've lost most of my great condition YAML files and a few in an old archive. If you'd like to help and contribute for config.ymls for plugins that require explanation in their configs, well now you can do that.

    .

    Just put a pull request on the MCYA.ML Repository.

    .

    We will make a list and you'll be on here if you can contribute!
